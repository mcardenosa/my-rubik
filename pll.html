PLL (Permutation of the Last Layer)

The last step. Seriously, you will be done after this.

The last step. Now we only need to exchange the pieces of the last layer without rotating them and we will have completed the cube. Leaving symmetries aside, after completing the OLL alogrithm, 14 different cases can be presented to us (including the case in which they are resolved).

As mentioned before, there are 14 cases if we do not count the symmetries. However, you should know all the cases with symmetrics included so that you can solve the cube using a single algorithm. Therefore, you will need to learn 21 cases. If you feel like 57 cases of OLL and PLL are too much for you, there is a 2 look PLL system, using only 6 algorithms. You can learn it here.
