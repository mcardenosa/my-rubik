OLL (Orientation of the Last Layer)

In this step you will be orienting the last layer stickers correctly

Now let's start with the bottom face. In order to make it more comfortable, we will rotate the whole cube so that the bottom face is up. Now what we have to do is get this face to have one single color. In other words, we have to orient each piece. Without counting symmetries, we can get 41 different cases (including the case in which all stickers are well oriented).

Now, as we said, there are 40 different situations that we can have in this step if we do not count symmetries. Each one of those cases can be solved with a single algorithm that will take you to a correctly oriented last layer.
Usually, people tend to like algorithms with double turns and fewer cube rotations, but this is a matter of taste and there are no rules. We will leave several algorithms for every case and it will be up to you to test a few of them and decide which one fits better your cube moves.
